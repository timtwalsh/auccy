# Auccy

Auccy is an opinionated Auctionator/TSM/Vanilla compatible Auction House side kick for buying Consumables and Gems.

### Modern Problems need modern solutions

The Vanilla auctionhouse was not built with Modern Megaservers in mind! There are too many listings and this means that traditional search methods break down.

I find myself often knowing what I want, but not knowing the exact name. (Yes I know it's a skill issue) The result is waiting for Auctionator or the Auction House to list hundreds of items that are not what I want. This is because I have to search for "Blood Garnet" or "Gem > Red > Common" (or worse if using vanilla search) instead of "Bold Blood Garnet".

### Auccy saves you time on common searches for Consumables and Gems!

The difference in search time is huge. Auctionator listing all 20 pages of Blood Garnets to get all listings takes about 1m 30s. Clicking through Auccy's menu takes about 5 seconds, and the search takes about 2 seconds - INCREDIBLE TIME SAVINGS!

### Auccy helps you quickly buy the gems and consumables you need.

Just click the item you want, and the Auccy will search the AuctionHouse and use your built in Vanilla/Auctionator/TSM interface to let you buy the items
